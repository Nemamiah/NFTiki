# NFTIKI SYSTEM – README (CARNAGE ANONYME)

## Projet :
🐾 Carnage interactif pour petits mutants et vieux enfants.  
Basé sur les **archives d’une chienne** devant une **porte bleue**.

## Genre :
NFT polymorphe. Audiovisuel. Écrit. Fantôme.

## Description :
Chaque NFTIKI est une **fragmentation codée** du Carnage.  
Un vestige. Une vision. Un glitch affectif.  
Les éléments s’imbriquent sans centre : son, image, voix, code, faille.

## Composants :
- 🎞 Séquences rituelles (vidéo floutée, format brut)
- 🎨 Figures dessinées par des mains anonymes
- 📖 Pages d’un livre pour mutants éveillés
- 🎤 Sons récoltés dans les interstices
- 💀 Métadonnées : {peur, amour, honte, boucle, seuil}

## Réseau :
- 🔗 Chaîne : Éthérée ou SubAlternative (faibles frais / accès libre)
- 🧾 Format : ERC-721 modulaire
- 🌀 Contrat : Non figé. Activable. Peut se réveiller.

## Licences :
- 🛠 Usage libre non-commercial (diffusion encouragée)
- 🔏 Noms effacés. Seules les vibrations comptent.

## Mécaniques :
- 🔁 Répétition rituelle : certains NFTs apparaissent cycliquement
- 🔓 Contenu à débloquer (en fonction de l’heure, du clic, ou du souffle)
- 🧬 Fusion possible : combinaisons aléatoires créant un nouvel Acte

## Objectifs :
- Activer une mémoire partagée sans nom
- Créer une boucle autonome de récits disloqués
- Lever de quoi nourrir la bête et continuer la fiction vivante


## Statut :
✅ Matériaux rassemblés  
🧪 En train de muter  
🎯 Mint initial : 13 fragments / premier seuil
